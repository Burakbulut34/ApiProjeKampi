🍴 ApiProjeKampi.WebApi
📌 Açıklama

Bu proje ASP.NET Core Web API kullanılarak geliştirilmiş bir restoran temalı örnek projedir.
Katmanlı mimari yapısı ile geliştirilmiş olup, CRUD işlemleri (ekleme, listeleme, güncelleme, silme) yapılabilmektedir.

🛠️ Kullanılan Diller ve Teknolojiler

C#

ASP.NET Core Web API

Entity Framework Core

SQL Server

AutoMapper

FluentValidation
